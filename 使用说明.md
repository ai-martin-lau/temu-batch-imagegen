# 📸 商品图批量出图 · 使用说明

**一句话**：把商品图放进 `input` 文件夹，双击一下，自动帮你出一套电商主图/套图。

---

## 一、第一次用（只需设置一次）

本工具靠 **codex CLI** 出图，所以电脑要先装好它并登录。

### 1) 安装 codex CLI

任选一种，装好后终端里能运行 `codex` 命令即可：

- **方式 ①：装 Codex 桌面应用（最简单，推荐非技术用户）**
  从 OpenAI 官方下载并安装 **Codex** 应用（Mac / Windows），它**自带 `codex` 命令行工具**，装好后命令行也就有了。

- **方式 ②：命令行安装（给技术同事）**
  - npm（需先装 Node.js）：`npm install -g @openai/codex`
  - 或 Homebrew：`brew install codex`
  > 包名/命令可能随版本变化，**以 OpenAI 官方文档为准**（github.com/openai/codex）。

### 2) 登录

打开「终端」运行：

```bash
codex login
```

按提示用你的 **codex / ChatGPT 账号**登录（走**订阅**，**不需要 API key、不额外花钱**）。装了 Codex 应用的话，在应用里登录一次也行。

### 3) 验证装好没

```bash
codex --version
```

能打印版本号（如 `codex-cli 0.x.x`）就说明 OK 了。

### 4) 升到最新版（建议）

本工具会自动用当前最强的模型出图，但**前提是你的 codex 别太旧**。第一次用前、或隔段时间，跑一下：

```bash
codex update
```

（装了 Codex 桌面应用的话，它一般会自己提示更新，点更新即可。）保持最新，出图才会用上最新最好的模型。

> 装好、登录一次，以后就不用再管了（偶尔更新一下更好）。

---

## 二、平时出图（三步走）

**1. 放图** 📥　把商品图放进 `input` 文件夹，两种方式：
　　- **一个商品一张图**：直接把图片（`.png` / `.jpg`）拖进 `input`。
　　- **一个商品有多张图（组图）**：在 `input` 里新建一个文件夹（名字就是这个商品名），把这个商品的多张照片都放进去。
　　放几个商品都行——**每个商品各出一套，互不干扰**。

**2. 双击** ▶️　双击 `批量出图.command`。
　　会弹出一个黑色窗口开始干活，**耐心等它跑完**（图越多越久）。
　　> 第一次双击若提示"无法打开"：右键点它 → 选「打开」→ 再点「打开」确认一次就好。

**3. 取图** 📤　跑完会自动打开 `output` 文件夹。
　　成品在 `output/batch-日期时间/图片名/` 里，命名 `01.png`、`02.png`……

---

## 三、改"出图要求"（最重要）

**要出几张、什么风格、什么尺寸、什么卖点——全写在 `prompt/默认.txt` 这个文本文件里。**

- 用文本编辑器打开 `prompt/默认.txt`，像写需求清单一样改（**纯中文写就行**）。
- 想给某个商品**单独定制**：新建一个**和图片名 / 文件夹名同名**的 txt。
  例：图片叫 `连衣裙.png`（或组图文件夹叫 `连衣裙/`），就建 `prompt/连衣裙.txt`，这个商品就用它。
- 没单独定制的图，统一用 `默认.txt`。

> 📂 参考 `example/` 文件夹——里面有一份真实的 prompt 和出好的 7 张图，**照着改最快**。

---

## 四、常见问题

| 现象 | 怎么办 |
|---|---|
| 双击没反应 / 被文本编辑器打开了 | 权限丢了。让技术同事在终端跑一次 `chmod +x 批量出图.command`；或直接终端里 `bash batch.sh` |
| 提示"没找到 codex" | 还没装或没登录 → 回到第一步 |
| 提示 input 没有图 | 先把图片放进 `input` 文件夹再双击 |
| 某张图效果/尺寸不满意 | 图像 AI 偶尔出偏，重新跑一次；或把 prompt 里的要求写得更明确 |
| 想多出 / 少出几张 | 改 `prompt/默认.txt` 里"输出 X 张"那句 |

---

## 五、小提示

- 出图是 AI 生成，**文字（尤其外语）偶尔会有错字，出图后请逐张过一眼**。
- 所有"出图要求"（尺寸、风格、卖点、要不要去 logo 等）都写在 prompt 里，**改要求只改 prompt**。
- 每次跑都会新建一个带**时间戳**的文件夹，**不会覆盖**上次的结果，放心多跑。
